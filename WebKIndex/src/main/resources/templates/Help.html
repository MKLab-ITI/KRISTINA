Help

Examples of making a query to KIndex.

index?query=baby sleep newspaper?title=Fabian Bähr freut sich auf den 2. Horber Neckar-Balloncup-Start am Mittwoch

Notes

Newspaper

  • How to call: /newspaper?title=Here_goes_the_title
  • Title may contain spaces, but the url string has to be URL Encoded (ex: “space” -> %20)
  • Article can be retrieved ONLY by title
  • Newspaper crawling takes place every 12 hours. Last articles may not be indexed yet.
  • Some articles in this newspaper are restricted from free access. Only publicly free articles are indexed (If a user can read an article in the browser, this article is publicly available!)
  • The response is in json format. The Content of the article is in “content” field
  • Source: http://www.tagblatt.de
  • Execute Newspaper Crawling

Document

  • How to call: /index?query=baby sleep
  • Query may contain spaces, but the url string has to be URL Encoded (ex: “space” -> %20)
  • The response is in json format. The Content of the webpage is in “content” field.
  • The passage is in “passage” field (For the current version, passage retrieval is not implemented yet, the passage provided is the 200 first characters of the document)

Passage Retrieval

  • Print score table for a query: /passagetable?query=baby
  • Print a list with top n documents of AllSegments index /getpassage?query=baby
  • Print score table for a WebAp Dataset: /webappassagetable
  • More query input options
  • Get info about the stored indices
  • Get available passage retrieval versions
  • Get indexed urls (ES)
  • Get scraped content from a url
